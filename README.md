# Robot-Telecontrol
This repo includes all material needed to reproduce the system implemented in the master thesis "A Wearable System for Robot Teleoperation with Haptic Feedback: Integrating Magnetic Tracking and EMG-Based Sensing" from January 2026. It is split into several subprojects that are meant to run in parallel for a full 7-DOF control.


For all the code needed for EMG sensing, model training and inference (2 rotational DOFs): **Myo_armband_tracking**


For all the code needed for the magnetometer matrix tracking magnet on the palm of the user's hand (3 transational DOFs): **Magnet Tracking**


For all the code needed for the robotic wearable exoglove and the C++ code of the user's station connected to the robot station: **Telecontrol v3.0 - Long Distance**


For the C++ code needed for the robot station controlling the robot in real-time: **Robot Station**
